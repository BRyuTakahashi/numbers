# Numbers

**Este projeto é uma landing page de um gerador de número aleatório podendo escolher o minímo e máximo e se repete ou náo o número**

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JS ES6+

### ✨ Funcionalidades

* CSS Animations
* Math.Random
* Array Functions
